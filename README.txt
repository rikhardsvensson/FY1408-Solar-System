==========================================================================
Directory Structure Overview
==========================================================================
	\.git				Git's unmentionables.
	
	\bin				The compiled library Irrlicht.dll. 
						The default output directory for the compiler.
	
	\irrlicht-1.8.2		The uncompressed bare bones needed to run 
						the Irrlicht engine. Also the entire compressed SDK 
						with samples and complete documentation.
						
	\resources			Textures, meshes, shaders, planetary data and so on.
